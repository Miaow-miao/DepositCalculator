#Описание проекта

Данное WPF-приложение предназначено для расчета дохода по вкладу.

Приложение поддерживает:
- расчет простых процентов
- расчет сложных процентов
- ввод исходных данных пользователем
- обработку ошибок ввода

---

#Функциональность

- Ввод суммы вклада
- Ввод срока вклада (в месяцах)
- Ввод процентной ставки
- Выбор типа начисления процентов:
  - простые проценты
  - сложные проценты
- Расчет итоговой суммы дохода
- Отображение результата
- Обработка некорректного ввода

---

#Архитектура проекта

Проект состоит из двух основных частей:

- **WPF-приложение (UI)**
- **Класс бизнес-логики (Calculator)**

Формулы расчета:
- Простые проценты:  
  `P * (1 + rate * t)`
- Сложные проценты:  
  `P * (1 + rate / 12) ^ months`
